## IES-SPECIFIC LEARNINGS

1. **P&L URL routing**: `/app/profitandloss` returns 404 on IES. Must navigate to `/app/standardreports` and click the dynamic PANDL link with `rptId` and `token` parameters.
2. **Company switcher**: Click entity name in header → dropdown shows Consolidated View + individual entities. URL params don't switch entities.
3. **JE form**: Uses `#hoveringLineWrapper` pattern. Input IDs are dynamic and change per row activation. Must re-query after row switches.
4. **AR requires customer**: When using Accounts Receivable account in JE, QBO requires a customer in the Name field. IC Customers from entity picker work.
5. **IC Customer naming**: Format is "Entity Name (Par./Ch.) (IC Customer)" — selectable from Name combobox dropdown.
